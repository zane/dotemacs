fundraising

The Obama for America Fundraising app leverages Square to enable individuals to make donations using their credit cards.

Installation

Install Homebrew.

Install the Android SDK and maven:

brew install android-sdk
brew install maven

Update your Android SDK

android update sdk --no-ui

Install the Google Maps libraries into your Maven repository

git clone https://github.com/mosabua/maven-android-sdk-deployer.git <path>
cd <path>
mvn install

Install the latest OFA fork of Robolectric into your Maven repository

git clone git@github.com:ofa/robolectric.git <path>
cd <path>
git checkout ofa
mvn install

Running the Tests

mvn test

Building

mvn install

Installing

mvn android:deploy

Troubleshooting

if build fails to find SetupTask

android update project -p . -t <target>

to get a list of available targets

android list targets
